# Campaign

This folder houses all campaign-specific content for *The Broken Harmonies* — material that exists at the level of session prep rather than world canon, and may or may not eventually be integrated into the main vault.

## Structure

The Campaign folder mirrors the root vault structure:

| Folder | Contents |
|--------|----------|
| `Campaign/Events/` | Campaign-specific events and session milestones |
| `Campaign/Factions/` | Factions unique to or detailed for the campaign |
| `Campaign/Governments/` | Political bodies relevant to campaign arcs |
| `Campaign/Items/` | Quest items, unique gear, named artifacts |
| `Campaign/Lore/` | In-world documents, handouts, letters |
| `Campaign/Magic/` | Campaign-specific magic rules, spells, rituals |
| `Campaign/Maps/` | Map art notes, style guides, keyed map files |
| `Campaign/People/` | NPCs, contacts, recurring characters |
| `Campaign/Religions/` | Campaign-relevant cults, sects, denominations |
| `Campaign/Species/` | Species variants or subtypes specific to this campaign |
| `Campaign/Stories/` | Session recaps, in-world narratives, vignettes |
| `Campaign/Technology/` | Unique technomancy, experimental devices |
| `Campaign/World/` | Detailed locations: settlements, dungeons, regions |

## Canon vs. Campaign

- **World files** (`World/`, `People/`, etc.) = Permanent world lore, synced to wb.db
- **Campaign files** (`Campaign/`) = Session prep; may graduate to canon after play

When a campaign element is confirmed as world canon, move or copy it to the appropriate root folder and run `wb.py sync`.
